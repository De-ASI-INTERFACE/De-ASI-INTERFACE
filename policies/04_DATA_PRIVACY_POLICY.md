## 1. Purpose

This Data Privacy Policy describes how De-ASI-INTERFACE collects, processes, stores, and protects personal and on-chain data across its platforms, APIs, and monitoring systems.

---

## 2. Data We Collect

### 2.1 On-Chain Data
Publicly available blockchain transaction data (wallet addresses, transaction hashes, token balances) is indexed for operational and analytical purposes.

### 2.2 Off-Chain Data
- API usage metadata (request timestamps, endpoint access logs)
- Dashboard access logs from Grafana monitoring systems
- Email addresses provided voluntarily for communications

---

## 3. Data Use

Collected data is used exclusively for:
- Platform security monitoring and anomaly detection
- Trading performance analysis and system optimization
- Regulatory compliance and audit trail maintenance
- Communication with verified collaborators and partners

Data is **never sold, rented, or shared** with third parties outside of legally required disclosures.

---

## 4. Data Retention

| Data Type | Retention Period |
|-----------|----------------|
| Trading logs and PnL records | 7 years |
| Security incident logs | 5 years |
| API access logs | 90 days |
| Email communications | 3 years |
| On-chain indexed data | Indefinite (public blockchain) |

---

## 5. User Rights

Individuals may request access, correction, or deletion of their personal data by contacting: **privacy@de-asi-interface.io**

---

## 6. Security Safeguards

- All personal data encrypted at rest (AES-256) and in transit (TLS 1.3)
- Access restricted to authorized personnel only
- Data breaches reported to affected parties within 72 hours of discovery
